Algorithmic_trading

Общее введение

Данный проект направлен на визуализацию графиков котировок.

Функционал приложения

Выбор файла для отображения.
Визуализация алгоритма кубического сплайна и метода наименьших квадратов.
Вывод значений по определённой дате.
Очистка графиков.

Make targets

Таrgets:
 ⥗  all - Выполнение всех целей make
 ⥗  install - таргет для сборки приложения
 ⥗  unistall - таргет для удаления приложения
 ⥗  clean - очистка "мусорных" файлов
 ⥗  dvi - вывод паспорта проекта
 ⥗  dist - таргет для архивации приложенияT
